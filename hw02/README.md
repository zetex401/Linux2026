# backup.sh

## Описание
Скрипт создаёт резервную копию директории в формате .tar.gz и сохраняет её в папку ~/backups. После каждого запуска оставляются только последние 3 архива.

## Требования
В WSL2 должен быть включён systemd, чтобы работал cron.

Проверка:
ps -p 1 -o comm=

Ожидаемый результат:
systemd

## Настройка cron
Откройте crontab:

crontab -e

Добавьте строку для запуска каждый час:

0 * * * * /home/student/os-lab01/hw02/backup.sh /home/student/testdir

## Проверка работы
Проверьте, что создаются архивы:

ls -lt /home/student/backups

Также можно проверить задания cron:

crontab -l

## Отключение
Откройте crontab:

crontab -e

Удалите строку с backup.sh.

Если других задач нет, можно удалить весь crontab:

crontab -r
